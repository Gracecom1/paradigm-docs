## これは何か

Outbound Form Agent は、**Twenty CRM に登録された企業の問い合わせフォームを自動で送信する**ツールです。

操作はすべて **Twenty CRM** で完結。企業データを登録したら、Hermes に「エージェント実行して」と伝えるだけ。あとは自動で動きます。

## 操作モデル

| あなたがすること | 場所 | やること |
|------------------|------|---------|
| 企業を登録する | `twenty.paradigmjp.com` | 企業名・ドメイン・フォームURL を入力 |
| ステータスを「送信待ち」にする | `twenty.paradigmjp.com` | プルダウンで選択 |
| Hermes に指示する | Hermes チャット | `「エージェント --max 5 で実行して」` |
| 結果を確認する | `twenty.paradigmjp.com` | ステータスが自動更新されている |

**GUI ダッシュボードは必要ありません。** Twenty CRM が唯一の画面です。

## 処理の流れ

```
Twenty CRM（SSOT）
    │  Hermes から「実行」指示
    ▼
Docker コンテナ on Hetzner
    │  Twenty REST API で「送信待ち」企業を取得
    ▼
Stagehand v3 LOCAL（CDP Chrome）
    │  1 社ずつフォームにアクセス
    ├─ CAPTCHA 検出 → スキップ
    ├─ DeepSeek AI が営業文を生成
    ├─ Stagehand agent がフォーム入力・送信を自律実行
    └─ 結果を Twenty CRM に書き戻し
```

## 技術スタック

| コンポーネント | 技術 | 理由 |
|---------------|------|------|
| ブラウザ自動操作 | Stagehand v3 LOCAL + CDP | Playwright 不要。ブラウザを自前管理。外部サービス依存ゼロ |
| AI 文章生成 | DeepSeek API | 企業情報から適切な営業文を生成 |
| データ管理 | Twenty CRM REST API | 企業一覧・送信結果の SSOT |
| 実行基盤 | Docker on Hetzner CX22 | オンデマンド起動・処理後即終了 |
| 操作インターフェース | Hermes チャット | 「実行して」と言うだけ |

## 主な特徴

| 特徴 | 説明 |
|------|------|
| **Twenty SSOT** | 企業データは Twenty CRM だけ。ダッシュボード不要 |
| **Hermes 操作** | 「--max 5」と伝えれば Docker が起動して処理 |
| **Stagehand v3** | CDP 直結。Playwright レイヤー不要。bot 検出回避 |
| **CAPTCHA 自動スキップ** | reCAPTCHA/hCaptcha/Cloudflare を検出しスキップ |
| **並列 8 社同時** | 1 社約 20 秒。500 社でも約 20 分で完了 |
| **エラー耐性** | 最大 3 回リトライ。それでも駄目なら安全にスキップ |
| **日本語完全対応** | Twenty CRM の日本語ステータス値に対応 |

## Hermes からの実行方法

```
# 1. Twenty CRM で企業を「送信待ち / 未対応」に設定

# 2. Hermes チャットで実行指示（例）:
「エージェントを --max 5 --dry-run でテスト実行して」
「エージェントを --max 10 で本番実行して」
「エージェントを --company "株式会社ABC" --single で 1 社だけ実行して」

# 3. Twenty CRM で結果確認
https://twenty.paradigmjp.com
→ ステータスが「送信済み」「手動確認」等に自動更新されている
```

## 結果ステータス

| ステータス | 意味 | あなたのアクション |
|-----------|------|-------------------|
| `送信待ち / 未対応` | 処理待ち | 何もせず待つ |
| `送信済み / 未対応` | 送信成功 | 企業からの返信を待つ |
| `手動確認 / 未対応` | 自動送信失敗 | エラーを見て手動対応 |
| `ブロック / 未対応` | CAPTCHA 等でブロック | メール等他手段で連絡 |
