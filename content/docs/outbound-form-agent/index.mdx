## これは何か

Outbound Form Agent v3 は、**複数の無料ソースから SMB ドメインを自動収集し、Twenty CRM に登録・審査・フォーム自動送信する** CLI ツールです。

v3 の新機能:
- **マルチソース収集**: crt.sh / urlscan.io / Whoxy / CZDS / Tranco — 5 ソース並列
- **品質ゲート**: DeepSeek 審査スコア < 30 & CMS 未検出 → CRM 登録除外
- **フォーム URL 自動検出**: 17 パス HEAD プローブ
- **MassDNS + Wappalyzer**: DNS 解決 + CMS 技術検出
- **重複防止**: 既存 CRM ドメインを自動スキップ

## 操作モデル

| あなたがすること | 場所 | やること |
|------------------|------|---------|
| ドメイン収集を指示 | Hermes チャット | `「--collect-sources us --max 20 で実行して」` |
| フォーム送信を指示 | Hermes チャット | `「--max 5 で実行して」` |
| 結果を確認 | `twenty.paradigmjp.com` | paradigmSalesStatus 列を確認 |

## 処理の流れ

### マルチソース収集
```
Hermes 指示
  → crt.sh / urlscan.io / Whoxy / CZDS / Tranco から並列収集
  → 重複除去
  → MassDNS で DNS 解決
  → HTTP スキャン + Wappalyzer CMS 検出
  → フォーム URL 自動検出 (17パスHEADプローブ)
  → DeepSeek 企業審査スコアリング
  → 品質ゲート (スコア30未満 & CMS無し→スキップ)
  → Twenty CRM に登録
```

### フォーム自動送信
```
Twenty CRM から「送信待ち」企業を取得
  → Stagehand LOCAL (Playwright Stealth + CDP Chrome)
  → CAPTCHA 検出 → 即スキップ
  → DeepSeek 営業文生成
  → フォーム入力・送信
  → CRM に結果書き戻し
```

## 技術スタック

| コンポーネント | 技術 |
|---------------|------|
| ブラウザ自動操作 | Stagehand LOCAL + Playwright Stealth |
| AI 文章生成/審査 | DeepSeek API |
| DNS 解決 | MassDNS (自前ビルド) |
| CMS 検出 | Wappalyzer |
| データ SSOT | Twenty CRM REST API |
| レート制限 | 内蔵 RateLimiter (crt.sh 2s / urlscan 5s / Whoxy 10s) |
| 実行基盤 | Docker on Hetzner CX22 |
| 操作 I/F | Hermes チャット |
