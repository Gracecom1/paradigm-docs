## 前提条件

**あなた自身の PC にインストールするものは何もありません。**

| 必要なもの | 説明 |
|-----------|------|
| Twenty CRM アカウント | `twenty.paradigmjp.com` にログインできること |
| Hermes チャット | エージェントへの実行指示を出すインターフェース |
| インターネット接続 | 上記 2 つにアクセスできれば OK |

---

## 操作の流れ

### マルチソース収集（新規リード獲得）

```
Hermes チャット
    │  「--collect-sources us --max 20 で実行して」
    ▼
Hetzner サーバー
    │  5ソースからドメイン並列収集 → DNS → HTTP → 審査 → CRM
    │  コンテナ自動終了
    │
結果確認
    └── Twenty CRM 画面 → 新規企業が paradigmSalesStatus 付きで登録される
```

### フォーム自動送信（CRM の企業に送信）

```
Hermes チャット
    │  「--max 5 で実行して」
    ▼
Hetzner サーバー
    │  Twenty CRM から「送信待ち」企業を取得 → Stagehand でフォーム送信
    │  結果を CRM に書き戻し
    │
結果確認
    ├── Twenty CRM 画面 → paradigmSalesStatus が変化
    └── 「--history を見せて」
```

## Hermes への指示例

### マルチソース収集

```bash
# 米国ドメインを 20 件収集
「--collect-sources us --max 20 で実行して」

# 日本ドメインを 50 件収集
「--collect-sources jp --max 50 で実行して」

# ゾーンファイルから収集 (.com ドメイン)
「--zone com --max-scan 100 で実行して」
```

### フォーム送信

```bash
# 5 社だけ本番送信
「--max 5 で実行して」

# 手動確認企業も含めて強制実行
「--max 5 --force で実行して」

# 安全確認のみ（送信なし）
「--dry-run で実行して」

# 単一 URL テスト
「--single-url https://example.com/contact --company "Example" でテストして」
```

### 履歴確認

```bash
# 送信履歴
「--history を見せて」

# 履歴サマリ
「履歴サマリを見せて」
```

### 管理

```bash
# 状態確認
「エージェントの状態を確認して」

# 再ビルド
「エージェントを再ビルドして」
```

## 結果の確認方法

### Twenty CRM
1. `twenty.paradigmjp.com` を開く
2. 企業一覧で以下を確認:

| フィールド | 意味 |
|-----------|------|
| paradigmSalesStatus | 収集状態 (送信待ち/未診断/カルテ生成中) |
| paradigmDataSources | 収集元ソース (crt.sh, urlscan.io, ...) |
| paradigmDataStatus | データステータス (multi_source_collected) |
| paradigmLastError | スコア・CMS・フォーム検出詳細 |

## トラブルシューティング

### 全社スキップされる (収集)
- DeepSeek スコアが 30 未満かつ CMS 未検出 → 品質ゲートで除外
- `--max` を増やす、または別のリージョンを試す

### 全社スキップされる (フォーム送信)
- Twenty CRM に「送信待ち / 未対応」かつ paradigmFormUrl ありの企業が必要
- マルチソース収集で formUrl が自動検出されていれば OK

### DeepSeek API エラー
- 残高・API キーの問題。Hermes に「DeepSeek API の状態を確認して」と伝える

### CAPTCHA で全滅 (フォーム送信)
- 多くの企業サイトが reCAPTCHA/hCaptcha を導入
- 非 CAPTCHA サイトでは送信可能
- hCaptcha/Turnstile は Stagehand が即検出→スキップ

### コンテナ起動失敗
- Hermes に「エージェントを再ビルドして」と伝える
