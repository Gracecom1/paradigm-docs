## Node.js 環境

```bash
# Node.js 22+ 推奨
node --version
```

## リポジトリのクローン

```bash
git clone https://github.com/Gracecom1/outbound-form-agent.git
cd outbound-form-agent
npm install
```

## 環境変数

`.env` ファイルを作成：

```bash
# .env ファイルに設定
TWENTY_BASE_URL=https://twenty.paradigmjp.com
TWENTY_API_KEY=your_jwt_token
DEEPSEEK_API_KEY=sk-your_key
DEEPSEEK_MODEL=deepseek-chat
LOG_LEVEL=info
```

## ビルド

```bash
npm run build
```

## 動作確認

```bash
# ドライラン（送信なし）
node dist/index.js --dry-run

# 単体テスト（1社のみ）
node dist/index.js --single

# 全件処理
node dist/index.js
```

## Docker セットアップ

### Homebrew + Colima（Apple Silicon）

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install docker colima
colima start --cpu 4 --memory 4
```

### ワンコマンドデプロイ

```bash
bash deploy.sh           # 本番実行
bash deploy.sh --single  # 単体テスト
bash deploy.sh --cron    # 30分間隔 cron
```
