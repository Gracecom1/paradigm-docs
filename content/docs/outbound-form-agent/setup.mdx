## 前提条件

| 必要なもの | 確認コマンド | 最小バージョン |
|-----------|-------------|--------------|
| Node.js | `node --version` | 22.x 以上 |
| npm | `npm --version` | 10.x 以上 |
| Git | `git --version` | 任意 |

### Node.js が入っていない場合

**macOS**:
```bash
brew install node@22
```

**Ubuntu/Debian**:
```bash
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs
```

インストール後、ターミナルを再起動して確認:
```bash
node --version  # v22.x.x と表示されれば OK
npm --version   # 10.x.x と表示されれば OK
```

## ステップ 1: リポジトリの取得

```bash
cd ~
git clone https://github.com/Gracecom1/outbound-form-agent.git
cd outbound-form-agent
```

確認: `ls` で `README.md  package.json  src/  deploy.sh  Dockerfile` が表示されれば成功。

---

## ステップ 2: 依存パッケージのインストール

```bash
npm install
```

初回は 30 秒〜1 分かかります。

### よくあるエラー

| エラー | 原因 | 解決方法 |
|--------|------|---------|
| `npm: command not found` | Node.js 未導入 | 前提条件を参照 |
| `EACCES` | npm 権限不足 | `sudo npm install` は非推奨。[npm 公式の権限修正手順](https://docs.npmjs.com/resolving-eacces-permissions-errors-when-installing-packages-globally) を参照 |
| `node-gyp` エラー | C++ ビルドツール不足 | macOS: `xcode-select --install`, Ubuntu: `sudo apt install build-essential` |

## ステップ 3: 環境変数の設定

```bash
cp .env.example .env
```

`.env` を開いて各項目を設定:

```bash
TWENTY_BASE_URL=https://twenty.paradigmjp.com
TWENTY_API_KEY=eyJhbG...
DEEPSEEK_API_KEY=sk-...
DEEPSEEK_MODEL=deepseek-chat
LOG_LEVEL=info
```

### API キーの取得

**Twenty CRM**: 管理画面 → Settings → API → Generate Key

**DeepSeek**: [platform.deepseek.com](https://platform.deepseek.com) → API Keys

---

## ステップ 4: ビルド

```bash
npm run build
```

確認: `ls dist/` で `index.js` が表示されれば成功。

---

## ステップ 5: 動作確認

```bash
# 安全なドライラン（送信なし）
node dist/index.js --dry-run

# 1 社のみテスト送信
node dist/index.js --single

# 全件処理（本番）
node dist/index.js
```

Tweny CRM の `paradigmSalesStatus` が「送信待ち / 未対応」→「送信済み / 未対応」に変わっていれば成功。

---

## ステップ 6: cron で定期実行

```bash
bash deploy.sh --cron
```

手動設定:
```bash
# crontab -e で追加
*/30 * * * * cd /path/to/outbound-form-agent && node dist/index.js >> /var/log/ofa.log 2>&1
```

## トラブルシューティング

### Twenty CRM に接続できない
1. `TWENTY_BASE_URL` が正しいか確認（末尾 `/` 不要）
2. `curl -I $TWENTY_BASE_URL` でサーバー応答確認
3. API キーを再発行

### フォームが見つからない
企業の Web サイトにお問い合わせフォームがない場合、自動スキップ。Twenty CRM の `paradigmLastError` に詳細が記録されます。

### DeepSeek API エラー
[platform.deepseek.com](https://platform.deepseek.com) → Billing で残高確認。
