操作はすべて Hermes チャット経由。コマンドラインや Git 操作は一切不要です。

## Hermes チャット指示一覧

### 基本

| 指示 | 説明 |
|------|------|
| `エージェントを --dry-run で実行して` | 送信せずフォーム検出・解析だけ確認 |
| `エージェントを --single で実行して` | 1 社のみテスト送信 |
| `エージェントを --max 5 で実行して` | 最大 5 社を本番送信 |
| `エージェントを実行して` | 全件処理 |

### 絞り込み

| 指示 | 説明 |
|------|------|
| `エージェントを --company "企業名" で実行して` | 特定企業のみ（部分一致） |
| `エージェントを --max 10 --single でテスト実行して` | 10 社をシングルスレッドで処理 |

### 管理

| 指示 | 説明 |
|------|------|
| `agent-runner の状態を確認して` | Webhook の稼働状態チェック |
| `agent-runner を再起動して` | Webhook の再起動 |
| `前回の実行結果を確認して` | Docker の終了ログを確認 |

## 全オプション一覧

| オプション | 値 | デフォルト | 説明 |
|-----------|-----|----------|------|
| `--dry-run` | — | — | 実際の送信はせず検出のみ。安全確認用 |
| `--single` | — | — | 1 社だけ処理。初回テスト用 |
| `--max <n>` | 数字 | 全件 | 最大処理数 |
| `--company <name>` | 文字列 | — | 特定企業名でフィルタ（部分一致） |
| `--concurrency <n>` | 1〜8 | 8 | 同時並列数 |
| `--timeout <ms>` | ミリ秒 | 120000 | 1 社あたりタイムアウト |
| `--log-level` | debug/info/warn/error | info | ログ詳細度 |

## 内部的に実行される Docker コマンド

以下は Hermes が内部的に実行するコマンド（参考。あなたが打つ必要はありません）。

```bash
# ドライラン
docker run --rm --env-file /opt/outbound-form-agent-v2/.env \
  --network host outbound-form-agent:v2 --dry-run

# 本番 5 社
docker run --rm --env-file /opt/outbound-form-agent-v2/.env \
  --network host outbound-form-agent:v2 --max 5

# 1 社テスト
docker run --rm --env-file /opt/outbound-form-agent-v2/.env \
  --network host outbound-form-agent:v2 --single

# 特定企業
docker run --rm --env-file /opt/outbound-form-agent-v2/.env \
  --network host outbound-form-agent:v2 --company "株式会社ABC"
```

## エージェントの内部動作

```
Hermes 指示
  → Agent Runner Webhook (:4444) に POST
  → systemd が docker run を実行
  → Twenty CRM から企業取得
  → Stagehand v3 LOCAL (CDP Chrome) でフォーム操作
  → DeepSeek API で営業文生成
  → フォーム送信
  → 結果を Twenty CRM に書き戻し
  → コンテナ自動終了
```

## 実行中の確認

Hermes に以下を伝える：
```
「実行中の Docker コンテナを確認して」
```

Hermes が `docker ps` で状態を確認する。

## ログの見方

### 正常時
```

[INFO] Completed 10/156 companies (6%)
```

### CAPTCHA スキップ
```

```

### タイムアウト
```
[WARN] Company "Slow Site": request timeout after 120s, retrying (attempt 2/3)
```


## エラー時の対処

| 現象 | 原因 | Hermes への指示 |
|------|------|----------------|
| 全社スキップ | CRM に「送信待ち」企業なし | Twenty CRM でステータス確認 |
| DeepSeek エラー | API キー・残高問題 | `DeepSeek API の状態を確認して` |
| コンテナ起動失敗 | イメージ破損 | `エージェントの Docker イメージを再ビルドして` |
| Webhook 応答なし | agent-runner 停止 | `agent-runner を再起動して` |
