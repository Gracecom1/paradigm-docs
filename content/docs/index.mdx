## 5 分で始める

```bash
# 1. Outbound Form Agent を入手
git clone https://github.com/Gracecom1/outbound-form-agent.git
cd outbound-form-agent

# 2. セットアップ
npm install
cp .env.example .env
# → .env に Twenty CRM の JWT と DeepSeek API キーを入力

# 3. ビルド
npm run build

# 4. ドライランで動作確認（送信なし）
node dist/index.js --dry-run

# 5. 1 社テスト送信
node dist/index.js --single
```

> 詳しい手順は [セットアップガイド](/docs/outbound-form-agent/setup) を参照。

## 日次運用の流れ

毎日やることはこれだけです:

```bash
# 1. コードを最新に
cd ~/outbound-form-agent && git pull && npm install && npm run build

# 2. 実行
node dist/index.js

# 3. Twenty CRM で結果を確認 → https://twenty.paradigmjp.com
```

> cron で完全自動化する方法は [CLI ツール群](/docs/cli-tools) を参照。
